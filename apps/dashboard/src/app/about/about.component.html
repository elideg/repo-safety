How to View All One Hungry Mind Repo Vulnerabilities

To communicate with the GitHub GraphQL API you'll need a OAuth Token.

In order to match the functionality of the GitHub GraphQL API use the following scopes when creating your OAuth Token

user public_repo repo repo_deployment repo:status read:repo_hook read:org read:public_key read:gpg_key

Once you generate your Auth Token with the right scopes, Copy & Paste it inside the Auth Token Input Box

You will only be able to see number of Vulnerabilities If you have the right perms in the GitHub Repo your searching
